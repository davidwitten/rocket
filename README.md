rocket
======

A state-of-the-art, revolutionary algorithmic language, it incorporates next-level syntax, inspiring innovation.
It is changing the world one program at a time.

Syntax
--------
      ! : output
      ? : input
      @ : load (with variable) (_variable to denote list)

      + : addition
      - : subtraction
      * : multiplication
      / : division
      % : modulus
      ^ : exponentiation

      $ : while-loop
      ~ : if-statement
      
      LIST:
      & : add element
      | : sorted list
